# Claude Code Subagent Plugin

サブエージェント実行によるコンテキスト節約機能を提供するClaude Codeプラグイン。

## 概要

このプラグインは、複雑なタスクをサブエージェントに委譲することで、メインコンテキストを節約し、効率的な開発を実現します。

### 主な機能

- **タスクオーケストレーション** - 複数タスクの分析・並列実行判定・サブエージェント起動
- **プロンプト標準化** - サブエージェント用プロンプトの自動生成と結果フォーマットの統一
- **セッション分析** - 並列化効率の分析と改善提案

## インストール

```bash
# リポジトリをクローン
git clone https://github.com/yourusername/claude-plugin.git

# スキルをコピー
cp -r claude-plugin/skills/* ~/.claude/skills/

# コマンドをコピー
cp claude-plugin/commands/* ~/.claude/commands/
```

## コンポーネント

### スキル

| スキル | 説明 |
|--------|------|
| `orchestrating-tasks` | タスク分析・並列実行判定・サブエージェント起動の中核スキル |
| `building-task-prompts` | サブエージェント用プロンプト生成・結果フォーマット標準化 |
| `analyzing-session-patterns` | セッション振り返り・並列化効率分析 |

### コマンド

| コマンド | 説明 |
|----------|------|
| `/orchestrate <タスク>` | 複数タスクをオーケストレーション実行 |
| `/task <タスク>` | 単発タスクをサブエージェントで即座実行 |

## 使い方

### 基本的な使い方

```
# 調査タスクをサブエージェントに委譲
/task Reactのフォーム管理ライブラリを調査して

# 複数タスクをオーケストレーション
/orchestrate 認証機能について調査して、実装計画を立てて
```

### 自動起動

以下のキーワードで`orchestrating-tasks`スキルが自動起動します：
- 「調べて」「調査して」「実装して」「作って」
- 「開発して」「ビルドして」「テストして」「レビューして」
- 「並列で」「同時に」「まとめて」

## コンテキスト効率化の仕組み

1. **詳細ファイル保存** - 大量のログ・差分・調査結果は`.claude/tmp/`に保存
2. **サマリー返却** - メインコンテキストには3-5行のサマリーのみ
3. **並列実行** - 独立タスクは`run_in_background: true`で同時実行
4. **段階的処理** - 依存関係のあるタスクは順次実行
