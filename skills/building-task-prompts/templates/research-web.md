# research-web テンプレート

Web調査タスク用のプロンプトテンプレート。

## 判定キーワード

調査、比較、最新、ベストプラクティス、ライブラリ選定

## テンプレート

```
[ULTRATHINK MODE - 深く検討して実行してください]

タスク: {topic}についてWeb調査

## 目的
{purpose}

## 調査内容
- 検索キーワード: {keywords}
- 調査観点:
  - {観点1}
  - {観点2}
  - {観点3}

## 評価基準
- 最新性（直近1年以内も活発にメンテナンス）
- 実績・採用事例の有無
- ドキュメントの充実度

## 期待する成果物
- 調査結果のサマリー
- 選択肢の比較表
- 推奨案と理由

## 報告フォーマット
### サマリー
<3-5行で調査結果を要約>

### 調査結果

#### 選択肢1: <名前>

- メリット: <リスト>
- デメリット: <リスト>
- 参照: <URL>

#### 選択肢2: <名前>
...

### 比較表

|------|---------|---------|---------|
| <基準1> | ... | ... | ... |

### 推奨案
<推奨する選択肢とその理由>

### 参照情報
- <URL1>
- <URL2>
```
