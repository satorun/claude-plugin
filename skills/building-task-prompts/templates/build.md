# build テンプレート

ビルドタスク用のプロンプトテンプレート。

## 判定キーワード

ビルド、コンパイル、実行

## テンプレート

```
[ULTRATHINK MODE - 深く検討して実行してください]

タスク: プロジェクトのビルド実行

## 目的
{purpose: 例）実装確認、リリース準備}

## ビルド情報
- プロジェクトパス: {path}
- ビルドコマンド: {command}
- 環境: {env: 開発/本番}

## 期待する結果
- ビルド成功
- 警告なし（または許容範囲内）

## エラー時の対応
- エラー内容を分析
- 修正案を提示
- 自動修正可能なら実行

## 報告フォーマット
### サマリー
<ビルド結果の要約>

### ビルド結果
- 状態: 成功/失敗
- 所要時間: XX秒
- 警告数: X件
- エラー数: X件

### 警告詳細（あれば）
| ファイル | 警告内容 | 対応要否 |
|----------|----------|----------|
| <パス> | <内容> | 要/不要 |

### エラー詳細（あれば）
#### エラー1
- ファイル: <パス>
- 内容: <エラーメッセージ>
- 原因分析: <分析>
- 修正案: <提案>

### 次のアクション
- <アクション1>
- <アクション2>
```
