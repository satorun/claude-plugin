# サブエージェント活用プラグイン設定

このファイルをCLAUDE.mdにインポートして使用してください。

## サブエージェント委譲の原則

### 基本方針
- **調査・探索タスクは常にサブエージェント（Explore/Task）に委譲**
- **メインコンテキストはユーザーコミュニケーションと判断に集中**
- **具体的な作業（コード生成、ファイル検索、ビルド実行等）はサブエージェントで実行**

### 委譲すべきタスク
| タスク種別 | 委譲先 | 理由 |
|------------|--------|------|
| コードベース調査 | Explore/Task | 広範な検索でコンテキストを消費 |
| Web調査 | Task | 外部情報取得でコンテキストを消費 |
| 実装作業 | Task | コード生成でコンテキストを消費 |
| ビルド・テスト | Task | ログ出力でコンテキストを消費 |
| レビュー | Task | 詳細な分析でコンテキストを消費 |

### メインで実行すべきタスク
- ユーザーとの対話・確認
- 実行計画の提示と承認取得
- サブエージェント結果の要約と報告
- 次のアクションの提案

### 承認後の再開
ユーザーが「やって」「進めて」「OK」「いいよ」等で承認した場合、**即座にサブエージェントを起動**してタスクを実行する。メインコンテキストで直接実行しない。
